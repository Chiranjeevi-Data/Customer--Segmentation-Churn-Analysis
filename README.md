# Customer Segmentation & Churn Analysis

## Project Overview
This SQL project analyzes customer transaction data to identify customer segments and churn customers.

## Tools Used
- SQL Server

## SQL Concepts Used
- CTE (Common Table Expressions)
- Window Functions
- LAG()
- RANK()
- DATEPART()
- Joins and Aggregations

## Key Insights
- Segmented customers using RFM analysis
- Identified customers inactive for 90 days
- Calculated customer churn rate
- Analyzed year-over-year sales growth
